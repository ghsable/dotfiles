# dotfiles
[dotfiles](https://ghsable.github.io/dotfiles/) is my Dotfiles + Scripts.

    dotfiles
    ├── .??*   : dotfiles(linux)
    ├── doc
    │  └── *.md   : documents
    └──bin
        ├── *.sh   : scripts(linux)
        ├── *
        │  ├── *.sh  : scripts(linux)
        │  └── *     : config files(linux)
        └── install_<OS_NAME>
            ├── .??*  : dotfiles
            ├── *.sh  : scripts
            ├── *.txt : data
            └── *
                ├── *.sh : scripts
                └── *    : config files
